Objetos en JavaScript - Lo esencial

1. ¿Qué es un Objeto?

Un objeto es una estructura que te permite almacenar información relacionada. Cada objeto puede contener propiedades (datos) y métodos (funciones).

Los objetos son fundamentales en JavaScript, porque representan cosas del "mundo real": personas, autos, usuarios, productos, etc. 

// Objeto simple
let persona = {
    nombre: "Ana",
    edad: 25,
    saludar: function() {
        console.log("Hola, soy Ana");
    }
};
Ejercicio: Crea un objeto llamado animal con propiedades tipo y edad, y un método hacerSonido() que imprima un sonido.

2. Crear Objetos

La forma más común de crear objetos es usando un objeto literal. Esta sintaxis es rápida, directa y muy utilizada. 

let coche = {
    marca: "Toyota",
    modelo: "Corolla",
    año: 2020
};
Ejercicio: Crea un objeto llamado computadora con propiedades: marca, ram y almacenamiento.

3. Acceder a Propiedades

Puedes acceder a los valores del objeto de dos maneras:

  • Notación de punto: sencilla y directa.
  • Notación de corchetes: útil cuando la clave tiene espacios o viene de una variable.
coche.marca;        // "Toyota"
coche["modelo"];    // "Corolla"
Ejercicio: Desde el objeto computadora del ejercicio anterior, muestra en consola la marca.

4. Modificar Objetos

Los objetos en JavaScript son modificables incluso después de crearse. 

coche.color = "Rojo";   // añadir propiedad
coche.marca = "Nissan"; // modificar propiedad
delete coche.año;       // eliminar propiedad
Ejercicio: Agrega una propiedad nueva llamada precio a tu objeto computadora.

5. Métodos de un Objeto

Un método es una función dentro de un objeto. Se usa para representar acciones del objeto. 

let persona = {
    nombre: "Luis",
    saludar() {
        console.log("Hola, soy " + this.nombre);
    }
};

persona.saludar();
Ejercicio: En tu objeto computadora, crea un método detallar() que muestre sus propiedades.

6. Objetos Anidados

Puedes guardar objetos dentro de otros objetos. Esto sirve para representar información compleja de forma clara. 

let usuario = {
    nombre: "Carlos",
    direccion: {
        ciudad: "Madrid",
        calle: "Sol 123"
    }
};

console.log(usuario.direccion.ciudad);

7. Recorrer un Objeto

Puedes usar for...in para recorrer las claves de un objeto. 

let persona = { nombre: "Ana", edad: 25 };

for (let clave in persona) {
    console.log(clave, persona[clave]);
}
Ejercicio: Recorre tu objeto computadora e imprime todas sus propiedades.

8. Métodos útiles de Object

Estas herramientas son muy comunes cuando manejas datos:

Object.keys(persona);   // ["nombre", "edad"]
Object.values(persona); // ["Ana", 25]
Object.entries(persona);// [["nombre","Ana"],["edad",25]]

9. Desestructuración

La desestructuración permite extraer valores del objeto fácilmente. 

let producto = { nombre: "PC", precio: 500 };

let { nombre, precio } = producto;
console.log(nombre, precio);

10. Clases (La Forma Moderna)

Las clases son "moldes" que permiten crear múltiples objetos del mismo tipo. 

class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    saludar() {
        console.log("Hola, soy " + this.nombre);
    }
}

let ana = new Persona("Ana", 25);
ana.saludar();
